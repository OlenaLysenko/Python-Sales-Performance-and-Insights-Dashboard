
Цей аналіз присвячений дослідженню продажів товарів компанії у різних категоріях та регіонах за певний період часу.  
Він включає:

- **Аналіз доходів та витрат** за категоріями товарів;
- **Визначення топ-5 категорій** за прибутком;
- **Загальний обсяг проданих одиниць** за категоріями та каналами продажів (online/offline);
- **Порівняння показників продажів** у різних країнах, регіонах та субрегіонах;
- **Вивчення часу між замовленням та відвантаженням (Aging)** для різних категорій, країн та регіонів;
- **Динаміка продажів у часі** за категоріями, країнами та регіонами;
- **Взаємозв’язок між прибутком та часом доставки товарів**;
- **Аналіз продажів за днями тижня** та виявлення закономірностей для топ-5 категорій.

Цей аналіз допомагає оцінити ефективність продажів, прибутковість різних товарів та планувати оптимізацію логістики та маркетингу.
